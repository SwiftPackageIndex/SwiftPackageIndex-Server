Ready for Swift 6

Swift 6 brings with it the ability to check your code for concurrency and data race issues. If switched on, the Swift compiler will produce errors where you could have data races.

For help migrating your code, see the Swift 6 language mode migration guide or the Swift 6 release blog post

To track the progress of the Swift package ecosystem, the Swift Package Index is running regular package compatibility checks across all packages in the index.
